How to Execute CLI Commands Using Python

By Harry Robson Posted Jan 20, 2023

When I first started learning Python, I was surprised to learn how easy it was to execute system commands from within my own Python program. It massively expanded my creative horizons, and that's why it's the first Python related post I will be making here. To follow along, you'll need to ensure that you have Python installed on your machine.

I'll be using the 'ping' command as an example. In case you are not familiar, the ping command is used to send an ICMP echo request to another node within a computer network. If the recipient node is live and accepting ping packets, it will reply with an ICMP echo reply. This is a handy command line utility if you want to check the reachability of specific nodes within a network. Run this example in your command prompt:

ping 127.0.0.1

You will get an output that looks something like this:

Reply from 127.0.0.1: bytes=32 time<1ms TTL=128

What this means is that the 127.0.0.1 (your loopback address) replied to your echo request and the reply packet was received within less than a millisecond. TTL is about how many more 'hops' the packet can do before it is dropped, but I'll not cover that topic right now. Now, let's automate it:

import subprocess
command = ["ping", "127.0.0.1", "-n", "3"]
execute_command = subprocess.run(command)

Above is code that will allow you to execute a command. Note how every argument in the command array is a separate string. You may also notice that I have added -n 3 to the end of the ping command. This means that 3 packets will be sent.

Now, you can run your program... It works! You should see a shell window appear (on Windows) and your command will be executed in front of your eyes. You're an automation genius!

Why You Need a Personal Website

By Harry Robson Posted Jan 18, 2023

It makes sense that my first post addresses the creation of my own personal website. My intention is to become the number 1 indexed page on Google for the term "Harry Robson". My reason for wanting to do this? One of the first things prospective employers, customers etc. will do is Google your name. What shows up? Without a personal website, you have little, to no, control over this.

My current situation (and most likely yours too) is that someone else will come up. You may not be listed at all! Whether it is right or not, my worry is that this leads to a certain level of disregard. In contrary, if a prospective customer was to search your name and you were the number one result, I would argue that the customer is going to trust you more - without ever having met you.

So how does it all work? Once you have created your website you are not automatically gifted the number 1 ranking on Google, Bing or DuckDuckGo! Not at all (unless you have a very unique name). More likely, you will be placed at the bottom of page 11, 12 or 45 for your own name! This is where SEO (search engine optimisation) comes in to play. You've probably heard of SEO, but what does it actually involve?

The main fields within SEO are: backlinking (where other sites link to yours); keyword optimisation (in which the author will integrate industry-relevant words into their site); technical SEO (how fast the page loads, mobile compatibility etc.) and the final that I will mention is domain age (how long your domain has been active). There are more factors, but these will be the focus of my efforts.

Are you convinced? Want to create your own personal website? It's really not that hard! Depending on how 'technical' you are, choose to either create a site using a website builder (such as Wix or Squarespace) or search on GitHub for a simple CSS blog template and edit it manually. This is what I have done, mainly as it allows me to keep costs and website size to a minimum! If you're following my method then you can then host on AWS for a few pounds per month and connect a custom domain!

I'll be posting some updates here detailling what worked/didn't once I'm more experienced with SEO, and hopefully hold the number 1 rank on Google for my name!
